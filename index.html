Assignment 1 - HTML comment

Comment out the second paragraph of the following HTML document 

        <h3>In this course we'll learn:</h3>
        <p>HTML & CSS</p>
        <p>Python</p>
        <p>Javascript</p>

Solution

Select a line of code then click "ctrl + /", this will comment out that line.
In this case, the second paragraph will look like this in the index.html page. 

<!-- <p>Python</p> -->

But, in the browser, "Python" will completely disappear. See example, below.

Assignment 3 - Hyperlink 2

Create a hyperlink to take you to another part of the same HTML document

Solution

In order to link to another section within the same html page, we need to use the
id name of the section we want to link to and add a hashtag, "#," infront of it as a value
of the href="...." attribute of the a hyperlink that want to link it to. 

          <section id="a1">
          <h2>Assignment 1 - HTML comment</h2>........</section>
          

          <a href="#a1">back to assignment 1 </a>
back to assignment 1

Assignment 4 - Hyperlink 3

Create a hyperlink to take you to another part of the same HTML document

Solution

Add target="_blank" attribute to the "a" tag. 

          <p>
            By clicking
            <a href="https://www.thisiscolossal.com/" target="_blank"
              >thisiscolossal,
            </a>
            google chrome will create a new tab directing you to the attached
            hyperlink.
          </p>

See example, below.

By clicking thisiscolossal, google chrome will create a new tab directing you to the attached hyperlink.

Assignment 5 - Article

Create an article element that has a title and a paragraph (HTML5 article
element represents independent item like a blog entry in an web document).

Solution

            <h2 class="article-title">Article Title</h2>
            <p class="article-paragraph">
              Lorem ipsum dolor sit amet consectetur adipisicing elit. Aliquid
              architecto recusandae eveniet voluptatum ad voluptatibus aperiam
              harum repudiandae aut nam dicta magnam saepe modi incidunt, illum
              debitis, quidem adipisci perferendis! Exercitationem impedit neque
              itaque harum asperiores! Assumenda eius perspiciatis quia? Fugit
              praesentium quasi, minima neque illum doloribus ducimus
              necessitatibus saepe?
            </p>
          </article>
